## [1000. # A+B](https://www.acmicpc.net/problem/1000)

### Problem

```
Given two integers A and B, calculate their sum.
```

### Input

```
The first line contains two integers A and B. (0 < A, B < 10)
```

### Output

```
Output one line of one integer, A+B.
```

### Example Input 1

```
1 2
```

### Example Output 1

```
3
```
